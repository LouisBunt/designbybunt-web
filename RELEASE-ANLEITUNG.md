# Louis Bunt Web – Release auf GitHub Pages

## Einmalig (ca. 5 Minuten)

1. Auf https://github.com einloggen (oder Account erstellen).
2. Neues Repository anlegen: **github.com/new**
   - Name: `louis-bunt-web` (oder `DEIN-USERNAME.github.io` für die kürzeste URL)
   - Public, ohne README
3. Auf der Repo-Seite: **"uploading an existing file"** klicken.
4. Den kompletten Inhalt des Ordners `louis-bunt-web` per Drag & Drop hochladen
   (alle Dateien + Unterordner: css, js, img, downloads, die HTML-Dateien, robots.txt).
   Alternativ: `louis-bunt-web-release.zip` entpacken und den Inhalt hochladen.
5. Commit-Button klicken.
6. **Settings → Pages**: unter "Build and deployment" Source = "Deploy from a branch",
   Branch = `main`, Ordner = `/ (root)`, Save.
7. Nach 1–2 Minuten ist die Seite live:
   - `https://DEIN-USERNAME.github.io/louis-bunt-web/`
   - bzw. `https://DEIN-USERNAME.github.io/` beim Spezial-Repo-Namen

## Updates später

Datei im Repo ändern/ersetzen (Website-Ordner → "Add file → Upload files") → Commit.
Die Seite aktualisiert sich automatisch.
